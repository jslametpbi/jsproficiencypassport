## GitHub Pages Deployment

1. Create a public repository named:

```text
jsproficiencypassport
```

2. Upload the extracted files directly to the repository root:

```text
index.html
styles.css
app.js
manifest.json
README.md
```

3. Go to **Settings → Pages**.
4. Choose **Deploy from a branch**.
5. Select **main** and **/root**.
6. Save.

The platform will be available at:

```text
https://jslametpbi.github.io/jsproficiencypassport/
```

Certificate verification demo:

```text
https://jslametpbi.github.io/jsproficiencypassport/#verify/JSPP-CERT-SAMPLE
```
